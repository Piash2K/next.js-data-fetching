# Next.js Data Fetching

A modern **Next.js 16** application demonstrating **data fetching** techniques with **MongoDB** as the database layer.  
Styled with **Tailwind CSS 4**, this project provides a solid foundation for building dynamic, data-driven web applications using the latest React (v19) and Next.js features.
